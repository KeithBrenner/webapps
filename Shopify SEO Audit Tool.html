Google Search Console

Index coverage, rankings, Core Web Vitals, schema validation

Free
0%
▼
Index Coverage
Total pages indexed vs. submitted
Check Coverage report. Note: how many pages are indexed, excluded, or erroring. Flag large gaps between sitemap URLs and indexed count.
Crawl errors (404s, 5xx server errors)
Pages & Coverage report → Error tab. Note any product/collection pages returning errors. Check if discontinued products have proper redirects.
Excluded pages — "Discovered, not indexed"
Large number here signals crawl budget or internal link issues. Check if important collection/product pages are excluded.
Shopify duplicate URL indexing Shopify
Check if /collections/*/products/* URLs are indexed alongside /products/* URLs. Shopify generates both — only canonical /products/* should be indexed.
Sitemap submitted and valid
Sitemaps → verify sitemap.xml is submitted. Shopify auto-generates /sitemap.xml. Note last crawl date and any errors.
Performance & Rankings
Top queries — impressions & clicks
Performance report → Queries. Identify top 20 branded vs. non-branded keywords. Note average CTR and position. Flag any queries with high impressions but low CTR (title/meta optimization opportunity).
Top pages — which pages get traffic
Performance → Pages. Note top 10 landing pages. Flag important product/collection pages with zero or near-zero clicks.
Click-through rate (CTR) vs. position benchmark
Positions 1–3 should have CTR above 10–15%. Low CTR on high positions = weak title tags or meta descriptions. Note any pages in positions 4–10 with good CTR (push candidates).
Mobile vs. desktop performance split
Performance → filter by Device. Check if mobile impressions/clicks are proportional. Large drops on mobile signal theme or UX issues.
Core Web Vitals
LCP (Largest Contentful Paint) — mobile & desktop
Core Web Vitals report. Good = under 2.5s. Note number of URLs flagged Poor vs. Needs Improvement. Shopify themes often fail here due to hero image loading.
INP (Interaction to Next Paint) — mobile
Replacement for FID. Good = under 200ms. Shopify apps and heavy JavaScript are common culprits. Note any pages flagged Poor.
CLS (Cumulative Layout Shift)
Good = under 0.1. Common Shopify causes: images without dimensions, font swaps, app-injected elements. Flag any pages above 0.25.
Schema & Rich Results
Rich results eligibility — product snippets
Enhancements section → Shopping/Product. Check for any warnings or errors. Note: eligibility doesn't guarantee display.
Breadcrumb rich results
Check Enhancements → Breadcrumbs. Most Shopify themes include breadcrumb schema but it's often incomplete or errors out on collection pages.
Security & Manual Actions
Manual actions — any penalties?
Security & Manual Actions → Manual Actions. Any issue here is critical and must be resolved first before other SEO work matters.
Security issues — hacked, malware flags
Security & Manual Actions → Security Issues. Any flag here is an immediate stop-everything issue. Note clean status as a positive.

Free

Setup & Configuration
GA4 property connected to Shopify
Confirm GA4 is firing via Shopify's native Google channel integration or GTM. Check real-time report for live data. Note if using Universal Analytics remnants.
Ecommerce tracking enabled Shopify
Admin → Data Streams → Enhanced Measurement. Verify purchase events are firing. Check Monetization → Ecommerce purchases report for data.
Conversion events configured
Admin → Events → check which events are marked as conversions. At minimum: purchase, add_to_cart, begin_checkout. Note any missing key events.
Internal traffic filter (IP exclusion)
Admin → Data Streams → Configure tag settings → Define internal traffic. Without this, client/agency visits inflate organic metrics.
Traffic & Acquisition
Organic search traffic % of total
Acquisition → Traffic acquisition. Note organic search share vs. direct, paid, social. Flag if direct is unusually high (often means dark social or misattribution).
Organic landing pages — where do users land?
Engagement → Landing pages → filter by Organic. Note which product/collection pages get organic entries. Flag pages with 0 organic entries but good GSC impressions.
Bounce/engagement rate on organic traffic
GA4 uses Engagement Rate (inverse of bounce). Under 40% engagement rate on product pages is a red flag. Note by page type: home, collection, product.
Ecommerce Performance
Organic search conversion rate
Monetization → Ecommerce purchases → segment by Organic. Note conversion rate and compare to site average. Organic typically converts better than social.
Cart abandonment rate
Funnel: add_to_cart → begin_checkout → purchase. Build an Exploration funnel report. Flag drop-off points. High abandonment at checkout = UX or trust issue.
Top revenue-generating pages from organic
Monetization → Ecommerce → filter by organic channel. Note top 5 revenue pages. These become the priority for schema, UX, and content improvements.

Schema markup validation for products, breadcrumbs, organization

Free
0%
▼
Product Schema (Test at least 3 product URLs)
Product schema — name, description, image
Run /products/[handle] through Rich Results Test. Confirm Product type detected. Verify name, description, and image fields populate correctly with no warnings.
Offer object — price, currency, availability
Critical for merchant listings. Verify price is numeric (not "$29.99"), priceCurrency is "USD", and availability is a schema.org URI (e.g. InStock). Missing = no price in SERP.
Brand field present in schema
Offer → brand → Organization or Brand type. Missing brand reduces merchant listing eligibility. Check /products/[handle].json for brand data availability.
GTIN / SKU / MPN present Shopify
GTINs are required for Google Shopping and improve merchant listing eligibility. Check if barcode field is populated in Shopify admin and passed to schema.
AggregateRating schema (if reviews exist)
If site has product reviews, verify ratingValue, reviewCount, and bestRating are present and valid. Check for review app compatibility (Okendo, Judge.me, Loox, etc.).
Protocol-relative or relative image URLs Shopify
Shopify CDN sometimes outputs //cdn.shopify.com (protocol-relative). Verify image URLs in schema start with https://. Protocol-relative images fail Rich Results validation.
Other Schema Types
Breadcrumb schema on collection & product pages
Test a collection page and a product page. BreadcrumbList should show correct hierarchy. Missing or broken breadcrumbs = no breadcrumb display in SERPs.
Organization / WebSite schema on homepage
Test the homepage. Verify Organization name, URL, and logo are present. WebSite schema enables the sitelinks search box. Low effort, high value for brand queries.

Site Audit — Technical Health
Overall Health Score
Site Audit → Overview. Note the health score percentage and number of Errors, Warnings, and Notices. Below 80% warrants investigation. Record baseline for comparison.
Broken internal links (3xx, 4xx)
Site Audit → Links → Broken. Note any product or collection pages with broken internal links. Broken links waste crawl budget and hurt UX.
Missing or duplicate title tags
Site Audit → On-page → Title issues. Note pages with missing, duplicate, or truncated (over 60 chars) titles. Shopify default titles often include store name suffix causing truncation.
Missing or duplicate meta descriptions
Site Audit → On-page → Meta description issues. Note count of missing/duplicate. Shopify doesn't auto-generate meta descriptions — they're often blank on product pages.
Images missing alt text
Site Audit → Images → Missing alt text. Note count and which pages. Product images without alt text miss keyword opportunity. Shopify often defaults to filename as alt text.
Slow pages (over 3s load time)
Site Audit → Performance. Flag pages over 3 seconds. Cross-reference with CWV data from GSC. Note if homepage and top product pages are affected.
Canonicalization issues Shopify
Site Audit → Indexability → Canonical. Look for pages where the canonical points elsewhere or is missing. Pay special attention to the /collections/ vs /products/ duplicate issue.
Backlink Profile
Domain Rating (DR) and total backlinks
Backlinks overview. Note DR score and total referring domains. For a new store, DR under 10 is typical. Flag any suspicious spikes in link acquisition.
Top referring domains — quality check
Backlinks → Referring domains. Note top 10 domains by DR. Check for toxic or spammy domains. Flag any link schemes or purchased links that may need disavow.
Lost backlinks (last 30 days)
Backlinks → Lost. Flag any high-DR links recently lost. Lost links to product pages can cause ranking drops. May indicate product URLs changed without redirects.
Organic Keywords
Total organic keywords ranking
Organic keywords overview. Note total count and how many are in positions 1–3, 4–10, 11–20. Flag branded vs. non-branded keyword ratio.
Top keyword opportunities (position 4–20)
Filter organic keywords by position 4–20 with meaningful volume. These are "quick win" candidates — small optimizations can push them to page 1. Note top 10.

AI Search Visibility

Brand mentions in ChatGPT, Perplexity, Google AI Overview, Gemini

AI Search
0%
▼
Brand Presence in AI Responses (Manual Checks)
ChatGPT brand mention test
Ask ChatGPT: "What is [brand name]?" and "Who makes [product type]?" Note whether the brand is mentioned, described accurately, or absent entirely. Copy response to notes.
Perplexity AI brand mention test
Search Perplexity for the brand name and key product category. Note: does the brand appear? Are sources cited that include the client's site? Perplexity shows citations — note if client URL appears.
Google AI Overview (SGE) appearance
Search Google for the brand name and main product keywords. Note if an AI Overview appears and whether the client's site is cited. Screenshot if applicable.
Competitor brand mentions in AI responses
Ask AI tools to recommend products in the client's category. Note which competitor brands appear and the client's absence or presence. This identifies the gap to close.
Technical Signals That Influence AI Visibility
About page — clear brand entity information
AI models build brand knowledge from structured content. Does the site have a clear About page with: who the brand is, what they make, founding story, location? This feeds entity understanding.
Wikipedia / Wikidata presence (if brand is large enough)
Search Wikipedia for the brand. AI models heavily weight Wikipedia as a source for brand knowledge. Not applicable for very small brands, but worth checking. Note if a page exists.
Organization schema with social profiles & contact
Organization schema on the homepage should include sameAs links to social profiles (Instagram, Facebook, LinkedIn). This helps AI models confirm brand identity across sources.
Third-party brand mentions (press, reviews, directories)
Search "[brand name] review" and "[brand name] mentioned". AI models build authority from external citations. Note count and quality of third-party mentions. No mentions = low AI brand authority.
FAQ / Q&A content targeting conversational queries
AI Overviews and LLM responses favor direct, question-answering content. Does the site have FAQ sections or blog posts that directly answer "how to", "what is", "best [product]" questions?

Shopify Platform Checks

Theme, URL structure, app bloat, redirect hygiene

Shopify
0%
▼
Theme & On-Page
Theme schema markup audit
Identify the theme (Settings → Theme → Actions → View code). Check if the theme uses product-schema.liquid or embeds schema inline in product.liquid. Note schema approach and any obvious gaps.
H1 tag audit on product & collection pages
View source or use browser inspector. Every product and collection page should have exactly one H1. Shopify themes sometimes use H2 for product names or have no H1 on collection pages.
Collection page content (thin content risk)
Check collection pages for descriptive text. Shopify collections often have only a product grid with no text content — this limits keyword targeting and can be flagged as thin content.
Product description quality
Review product descriptions on top 5 products. Are they manufacturer copy (duplicate content risk)? Are they SEO-friendly with natural keyword use? Are they substantive (100+ words)?
URL Structure & Redirects
Redirect audit — orphaned or chained redirects
Online Store → Navigation → URL Redirects. Check for redirect chains (A→B→C) and loops. Note total count. Large number of redirects may indicate past URL structure changes without cleanup.
robots.txt review
Visit /robots.txt. Shopify's default is generally fine but check: Are any important paths accidentally blocked? Is the sitemap referenced? Note any custom disallow rules.
Apps & Performance
Installed apps audit — performance impact
Apps → check active apps. Use PageSpeed Insights and toggle network tab to identify app scripts loading on storefront. Flag apps injecting scripts on every page that aren't needed globally.
PageSpeed Insights — mobile score
Run homepage and top product page through PageSpeed Insights. Note mobile Performance score. Below 50 is poor; 50–89 needs improvement; 90+ is good. Record LCP, TBT, CLS values.
Image optimization — format & compression
Check if product images are served as WebP (Shopify CDN does this automatically for modern browsers). Look for oversized images (over 2MB source). Check if lazy loading is implemented.
